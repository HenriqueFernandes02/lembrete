Opções de Envio

Se desmarcado, o robô vai simular o envio (não gasta créditos do Digisac).

Se desmarcado, nenhuma nota laranja será criada dentro do chamado.

Se habilitado, o robô buscará todos os números no Digisac que tenham a tag [ID_CLIENTE] e enviará mensagem para todos eles.

Se habilitado, o robô também enviará a mensagem para grupos do Digisac cujo nome contenha a tag [ID_CLIENTE]. Útil para notificar grupos de WhatsApp de empresas.
